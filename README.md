# FoodTracker
Team-Mitglieder: Dalia Salman, Beril Gümüs

Projektbeschreibung: Für unser Projekt haben wir das Thema „Food-Tracker“ gewählt. Dabei handelt es sich um eine Anwendung, mit der Nutzer ihre Kalorien sowie Nährwerte wie Kohlenhydrate und Proteine verfolgen können.

Nutzer können sich anmelden, ein Ziel festlegen (z. B. Abnehmen oder gesunde Ernährung) und erhalten darauf basierend ihren täglichen Kalorienbedarf. Außerdem können Lebensmittel und Mengen eingegeben sowie eine empfohlene Wasserzufuhr angezeigt werden.
